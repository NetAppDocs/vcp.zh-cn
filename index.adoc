适用于vCenter Server的NetApp Element 插件为VMware vSphere界面提供了一个插件、可用于管理和监控运行NetApp Element 软件的存储集群。

您可以使用本文档了解并使用此插件。本文档介绍了 4.6 及更高版本。有关此插件的旧版本，请参见 link:reference_earlier_versions.html["此页面"]。
